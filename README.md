# 🍏 Home Assistant: Mac OS 9 Platinum Edition
A high-fidelity, interactive recreation of the 1999 Macintosh OS 9 environment, optimized for Home Assistant dashboard control.

## 📁 Project Vision
This dashboard serves as the central hub for a Paoli-based smart home. Beyond standard controls, it is designed to eventually house a cute, isometric digital model of a classic single-story ranch house, blending retro computing nostalgia with modern automation.

## ✨ Current Features
- **Platinum Header:** 38px high-res menu bar with dynamic clock and Apple menu.
- **3D Control Strip:** A beveled sidebar containing Volume, Display, and Note Pad toggles.
- **Interactive Sticky Notes:** A functional Note Pad that toggles a high-contrast yellow "Sticky Note" in the center of the desktop.
- **Desktop Icons:** Anchored "Macintosh HD" and "Trash" icons that remain stable across different screen sizes.
