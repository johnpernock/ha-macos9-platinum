# 📜 Changelog

### [1.2.0] - 2026-03-26
#### Fixed
- **The "Black Box" Overlap:** Overcame Home Assistant theme overrides by switching Sticky Notes from `markdown` to `custom:button-card` with `!important` CSS flags.
- **Centering Victory:** Resolved "Top-Left Hugging" issue by implementing `place-self: center` on conditional cards.
- **Icon Stability:** Anchored Macintosh HD and Trash to a dedicated 2-row grid to prevent drifting.

### [1.1.0] - 2026-03-25
#### Added
- Implemented high-resolution (38px) header for modern displays.
- Added 3D beveled borders to the Control Strip.
- Scaled desktop icons to 72px "Large Icon" view.

### [1.0.0] - 2026-03-24
#### Initial
- Initial proof of concept for the Teal (#008080) desktop and Platinum UI.
